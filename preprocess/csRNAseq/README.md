# `preprocess/csRNAseq/`

The scripts in this folder generate a final set of TSSs and novel transcripts, as well as their quantifications in all samples.

## Software used

- HOMER v4.11.1
- STAR v2.7.9a
- samtools v1.15.1
- R v4.2.0
- BEDtools v2.29.2
- gffcompare v0.12.6
- gffread v0.12.7
- GFF3sort v1.0.0
- StringTie v2.1.6
